# Unity Audio Library
Sprite library but.. for audio

## How to Use
1. Create asset at /AudioLibrary/AudioLibraryAsset
2. Put Bgm/Sfx in each category, and give label for the audio clip
3. Get Clip by calling GetBGMAudio()/GetSFXAudio()
